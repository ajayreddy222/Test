# AKEEP-Gamification
AKEEP Unified Web Server for Online Educational Services

# Running the App

Make sure you have Node.js stuff installed:
```
cd frontend
npm install
```

Start the backend:

```
cd backend
mvn spring-boot:run
```

Start the frontend:

```
cd frontend
npm start
```

and connect to `https://localhost:3000` in a browser.
